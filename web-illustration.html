Web Illustration Project

Home Web Illustration

The goal of this project is to enhance sales by creating a Java-based web illustration platform that allowed the sales team to generate product illustrations with minimal input. The platform was designed to streamline the process, making it easier for sales agents to create personalized illustrations quickly and efficiently.

Requirement Gathering & Customization

Collaborated with various departments to gather information on the specific needs and challenges they faced while generating product illustrations.

Ensured the platform could be customized based on product type, as each product required unique inputs and calculations.

Verified that proposed solutions aligned with current regulations and business requirements.

Prefilling Input Data

Conducted extensive research into previously generated illustrations to identify the most commonly used values, allowing the platform to pre-fill default inputs, reducing manual effort for the user.

Implement self-service features to empower users to manage their policies independently.

User Interface Enhancement

Introduced sliders for key inputs, enabling users to select values via drag-and-drop functionality while still providing the option for manual input.

Developed the system to generate PDFs of the illustration and incorporated functionality to capture the customerâ€™s email, sending illustrations directly to the customer via email.

Collaboration with Actuarial Team

Worked closely with the actuarial team, who provided an Excel-based calculator that generated graphs and other values essential for the illustrations.

Coordinated with the development team to implement this Excel calculator into the web platform, ensuring the seamless integration of calculations with the illustration application.

Base Calculator Integration

The Excel-based calculator, provided by the actuarial team, was implemented by IT into the system, allowing the calculations to reflect accurately within the platform.

The tool generated visual graphs depicting contributions, investments, and benefits over time, providing a clear overview for the user.

Security Testing:

Conducted penetration testing (pentest) of the application to ensure compliance with data security standards and safeguard sensitive customer information.

Cross-Team Communication:

Encountered communication challenges as the development team, while proficient in understanding English, faced difficulty speaking it fluently. Overcame this by establishing clear, detailed documentation and holding regular meetings to ensure accurate communication and alignment on project goals.

Outcome

The platform significantly reduced the time required to generate product illustrations, allowing sales agents to create multiple versions and tailor them to individual customer needs on the spot.

The simplified process improved sales efficiency and customer engagement, as agents could quickly provide personalized product details.

The integration of sliders and default values enhanced user experience, allowing for rapid adjustments without manual data entry.

Tools & Technologies Used:

Java (for platform development)

JIRA and Confluence (for task tracking and documentation)

Excel (integrated calculator provided by the actuarial team)

PDF generation and email integration (for automated output)
